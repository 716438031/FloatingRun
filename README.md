# FloatingRun
### 悬浮窗执行 shell 指令

**一个悬浮窗执行shell指令的小程序**

这是把原项目从 eclipse 迁移到 Android Studio 的新分支。

支持 SDK 19（Android 4.4） - SDK 27 （Android 8.1）


### 使用场景:
玩某游戏的时候,经常需要重启游戏,所以打算使用am指令来进行重启...

命令：
`am start -S com.huanmeng.zhanjian2/org.cocos2dx.cpp.AppActivity`

然而需要 root ...
